# Smart Traffic Accident Severity Prediction

AI-based traffic accident severity prediction system using Machine Learning, SHAP, SMOTE, and Streamlit.

## Features
- Real-time severity prediction
- Explainable AI using SHAP
- Streamlit dashboard
- SMOTE balancing
- Risk analysis

## Technologies Used
- Python
- XGBoost
- Streamlit
- SHAP
- SMOTE
- Google Colab

## Severity Levels
- Level 1 → Low
- Level 2 → Moderate
- Level 3 → High
- Level 4 → Severe
